# Jomi Joma BD Rangpur

Professional Real Estate Marketplace built exclusively for the Rangpur Division context with seamless architectural scalability across Bangladesh.

## Project Vision & Disclaimers
*   **Marketplace Focus:** This application functions strictly as a property & assets advertising portal enabling direct connections between owners, buyers, agents, and developers.
*   **Non-Governmental Entity:** This platform is **NOT** a land record portal, **NOT** a land information directory, and has **NO** operational relationship with any government Khatian registry infrastructure.

## Technological Architecture Constraint Principles
*   **Runtime Framework:** Native HTML5, modern CSS3, and pure Vanilla JavaScript (ES6).
*   **Zero-Dependency Setup:** Completely free from external build environments, compilers, node packages (`npm`), or frameworks (React, Vue, Bootstrap, Tailwind, etc.).
*   **Hosting Context:** Runs native directly on the Edge network structure utilizing Cloudflare Pages static architecture.
*   **Future Adaptability:** Structured data and logic pipelines are abstracted natively inside standalone JSON entities within the `/data/` repository directory layout to easily scale up for Cloudflare Workers, D1 database storage systems, or RESTful API integrations down the road without modifying structural visual frameworks.
